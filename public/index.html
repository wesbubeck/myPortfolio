Crewify

A web app that is a project management software. It gives a project manager the ability to create job, include all the specs and customer data, assign a crew and other metrics. The data for the job is stored and is searchable by authorized personnel. Crew members have mobile access to a determined amount of data that can be live updated by the Project Manager. Once the job is complete the crew lead will mark the job completed and add job notes. All data is stored and Efficiency Metrics are generated and reported to the Project Manager

JavaScript | ES6 | Node.js | React | NPM | GitHub | Express | Axios | HTML 5 | Visual Code Studio | CSS 3 | BootStrap

Crypto Currency News Scraper

Like so many other people I am enthralled with Crypto Currencies and Block Chain, but I never have time to finish an article. So, I built a news scraper the will go to crypto and blockchain news sites and return the top new article. It will display the title, summary and a link to that article. Then you have the ability to save the article and they will be stored in a database. They will then be rendered when your return to the page. Saved articles can either be deleted or you can click notes, which will open a modal where you can write notes about the article that will also be stored in the database
